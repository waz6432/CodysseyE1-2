# 💡 파이썬 콘솔 퀴즈 게임
계층형 아키텍처(Layered Architecture)를 적용하여 설계된 사용자 참여형 퀴즈 프로그램입니다.

## 1. 프로젝트 개요
이 프로젝트는 Python을 기반으로 제작된 콘솔 인터페이스 퀴즈 게임입니다. 사용자는 등록된 문제를 풀거나 새로운 문제를 직접 등록할 수 있으며, 모든 데이터는 로컬 파일 시스템을 통해 영구적으로 보존됩니다.

## 2. 퀴즈 주제 선정 이유
* **주제:** [예: IT 상식 및 프로그래밍 기초 / 일본어 단어 / 일반 상식]
* **이유:** [예: 평소 관심 있는 분야의 지식을 체계적으로 복습하고, 단순한 코딩을 넘어 유지보수가 용이한 소프트웨어 구조를 실습해 보기 위해 선정하였습니다.]

## 3. 실행 방법
프로젝트는 별도의 외부 라이브러리 설치 없이 Python 3 환경에서 즉시 실행 가능합니다.

1.  **저장소 클론**
    ```bash
    git clone [본인의 저장소 URL]
    cd [저장소 폴더명]
    ```
2.  **프로그램 실행**
    ```bash
    # 프로젝트 루트에서 실행 (패키지 모드)
    python3 -m src.main
    ```

## 4. 데이터 관리

`state.json` 파일을 사용하여 애플리케이션의 모든 데이터를 통합 관리합니다. 별도의 복잡한 데이터베이스 설정 없이도 프로그램의 영속성(Persistence)을 보장합니다.

* **주요 역할:**
  * 기본 제공 퀴즈 및 사용자가 추가한 커스텀 퀴즈 데이터 보관
  * 사용자의 최고 점수 및 게임 플레이 히스토리 기록
* **특징:** 데이터가 루트 폴더의 단일 파일(UTF-8 인코딩)로 관리되므로, 파일 하나만 백업하면 언제 어디서든 이전 상태 그대로 이어서 플레이할 수 있습니다.

## 5. 기능 목록
- [x] **메뉴 인터페이스:** 번호 선택을 통한 직관적인 기능 제어
- [x] **퀴즈 풀기:** 저장된 문제 중 5개를 무작위로 출제하고 정답 확인
- [x] **퀴즈 추가:** 새로운 질문과 정답을 입력하여 데이터베이스에 등록
- [x] **퀴즈 목록 조회:** 현재 등록된 전체 퀴즈 리스트 확인
- [x] **최고 점수 기록:** 게임 종료 시 최고 점수를 비교하여 자동으로 업데이트
- [x] **데이터 영속성:** `state.json`을 통해 프로그램 종료 후에도 데이터 유지

## 6. 파일 구조
관심사 분리(SoC)를 위해 역할을 계층별로 나누어 구성하였습니다.

```text
.
├── README.md               # 프로젝트 상세 설명 문서
├── .gitignore              # Git 관리 제외 목록
├── state.json              # 데이터 저장 파일 (UTF-8)
└── src/
    ├── __init__.py         # 패키지 초기화 파일
    ├── main.py             # Entry Point (실행 진입점)
    ├── domain/             # 비즈니스 엔티티 (Quiz 클래스)
    ├── repository/         # 데이터 접근 계층 (JSON 파일 입출력)
    ├── service/            # 비즈니스 로직 계층 (게임 엔진 및 점수 계산)
    └── ui/                 # 프레젠테이션 계층 (콘솔 입출력)
